## 🚀 راه‌اندازی قدم به قدم

---

### 📱 قدم ۱ — ساخت ربات تلگرام (۲ دقیقه)

1. در تلگرام به **@BotFather** بروید
2. بنویسید: `/newbot`
3. نام ربات را وارد کنید (مثلاً: `ربات طالع‌بینی من`)
4. یوزرنیم بدهید (باید به `bot` ختم شود، مثلاً: `my_astro_bot`)
5. توکنی مثل این می‌گیرید: `7123456789:AAHxxxxxxxxxxxxx`
6. این توکن را کپی کنید ✅

---

### 🤖 قدم ۲ — دریافت کلید Google Gemini رایگان (۳ دقیقه)

1. به **[aistudio.google.com](https://aistudio.google.com)** بروید
2. با حساب Google وارد شوید
3. روی **"Get API Key"** کلیک کنید
4. **"Create API Key"** را بزنید
5. کلیدی مثل این می‌گیرید: `AIzaSyXxxxxxxxxxxxxxxxxxxxx`
6. کلید را کپی کنید ✅

> 🎁 **حد رایگان:** 1500 درخواست در روز — بیش از کافی!

---

### ☁️ قدم ۳ — هاست رایگان روی Railway.app (۵ دقیقه)

#### الف) آپلود کد

1. به **[github.com](https://github.com)** بروید و یک حساب رایگان بسازید
2. یک ریپازیتوری جدید بسازید (مثلاً `astro-bot`)
3. فایل‌های این پروژه را آپلود کنید (drag & drop)

#### ب) اتصال به Railway

1. به **[railway.app](https://railway.app)** بروید
2. با GitHub وارد شوید
3. روی **"New Project"** کلیک کنید
4. **"Deploy from GitHub repo"** را انتخاب کنید
5. ریپازیتوری `astro-bot` را انتخاب کنید

#### ج) تنظیم کلیدها در Railway

1. در پروژه Railway، روی **"Variables"** کلیک کنید
2. دو متغیر اضافه کنید:

```
TELEGRAM_BOT_TOKEN = توکن_تلگرام_شما
GEMINI_API_KEY     = کلید_gemini_شما
```

3. روی **"Deploy"** کلیک کنید ✅

> ربات شما الان برای همیشه و برای همه آنلاین است! 🎉

---

### 🖥️ روش جایگزین — اجرا روی کامپیوتر خودتان

اگر نمی‌خواهید از Railway استفاده کنید:

```bash
# نصب Python (اگر ندارید از python.org دانلود کنید)

# نصب وابستگی‌ها
pip install -r requirements.txt

# ساخت فایل .env
copy .env.example .env   # Windows
cp .env.example .env     # Mac/Linux

# فایل .env را با متن‌ویرا باز کنید و کلیدها را وارد کنید

# اجرا
python bot.py
```

> ⚠️ در این روش ربات فقط وقتی کامپیوتر شما روشن و متصل است کار می‌کند.
